Pixar Movie Trailers

Created by Brandon Yanofsky

Welcome to my Pixar Movie Trailers website. I built this for the final project of my Programming Foundations with Python class. The assignment is to use Python to create instances of a Movie class which stores movie title, trailer's YouTube URL, and poster image URL. It then runs through some given Python functions and outputs this website built within the Bootstrap framework.
